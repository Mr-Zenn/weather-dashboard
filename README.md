# 🌦️ Weather Dashboard

A real-time weather application that fetches live data using a public API.

## 🚀 Features
- Search any city worldwide
- Displays temperature, humidity, wind speed
- 5-day weather forecast
- Recent search history
- Clean and responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
